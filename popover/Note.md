# popover

## Advantages

1.No javascript needed (Reduce JavaScript Code)
2.press escape key to close
3.press on window screen to close
4.open as a popup

## Code:-

```html

    <div id="clicked" popover >
        <h1>Welcome to My Page</h1>
    </div>
    <button popovertarget="clicked" >Click me!</button>
```
